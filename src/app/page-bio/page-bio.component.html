My First Program in 2014

When I was in grade 6, Me and my classmates used to go to the computers lab. Everyone will sit on a computer and download Minecraft. And basically we play a single player world and see who survives on it. But since we were on the same local network. I was able to host my world on the LAN. And we would join and play on it. But half of the computers wasn't able to. Because they weren't on the same network. So, my friend asked me if I can create a server for us.

I didn't knew how to create a Minecraft server at all. It was a mysterious world to me. I watched tutorials on Youtube. I downloaded CraftBukkit and run it. I was able to connect to the server from my computer. I was so happy that I set the maximum players to 1,000,000 players.

I was confused about plugins and what do they do. But when I found out how they affect the behaviour of the server and the game itself. I was fascinated by what plugins can do after I found out.

Finally, that's the end of how I got into Minecraft servers. And the start of my nightmare with plugins.
